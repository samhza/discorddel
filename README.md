
# discorddel
- mass-delete Discord messages
- logs messages to sqlite
- downloads any attachments

`go install samhza.com/discorddel@latest`
```
Usage of discorddel:
  -archive string
    	Directory to log deleted messages in (default "./archive")
  -channel uint
    	Discord channel ID
  -guild uint
    	Discord guild ID
  -token string
    	Discord user token
```
